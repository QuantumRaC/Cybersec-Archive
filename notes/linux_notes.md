# Linux Notes & Resources

# Basic Commands

-           find
    -       find -name name.txt
    - outputs           /dir/name.txt
    - finds file with name / chosen extension

-           grep
    -       grep "word" name.txt
    - outputs selected entry with "word" highlighted

-           wc
    -       wc name.txt
    - outputs           100 name.txt
    - word count
